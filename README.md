# quick-env

- https://github.com/mouday/quick-env/

运维脚本整合，快速部署运行环境

下载解压

```bash
# yum 安装必要的依赖
$ bash main.sh init

# 查看版本号
$ qk version

# 安装2.3.9
$ qk install pyenv

# 安装3.7.0
$ qk install python

# 安装python2.7 pip
$ qk install pip

# 安装1.8.0
$ qk install nginx

# 安装4.2.5
$ qk install supervisor

# 安装8u351
$ qk install jdk

# 安装5.6.16
$ qk install elasticsearch

# 安装5.6.16
$ qk install kibana
```
